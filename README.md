# Customer Insights Report

End-to-end customer data cleaning and analysis project using **Python** and **pandas**.

---

##  Project Objective

Clean, transform, and analyze a messy customer dataset to generate **business-ready insights** and deliver clean outputs suitable for reporting and decision-making.

---

##  Dataset Description

The dataset contains customer information with multiple real-world data quality issues, including:

- Inconsistent name formatting
- Invalid and malformed email addresses
- Mixed country naming conventions
- Currency symbols and inconsistent numeric formats
- Multiple date formats
- Boolean values stored as text

---

##  Data Cleaning Process

### Text Cleaning
- Normalized customer names (case, spacing, underscores)
- Standardized email formats
- Cleaned and unified country names
- Converted activity status to boolean values

### Numeric Transformation
- Removed currency symbols and separators
- Converted spending values to numeric format
- Created annual customer value
- Calculated revenue-related KPIs

### Date Processing
- Unified multiple date formats into datetime
- Calculated days since last purchase
- Identified inactive customers based on business rules

---

##  Key Metrics & KPIs

- Total Revenue
- Average Spend per Customer
- Top Customer by Total Spend
- Total Number of Customers
- Inactive Customers (no purchases in the last 180 days)

---

##  Project Outputs

- Cleaned customer dataset (Excel)
- Cleaned customer dataset (CSV)
- Business KPI summary report (TXT)

All outputs are ready to be shared with non-technical stakeholders.

---

## Technologies Used

- Python
- pandas
- datetime
