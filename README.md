# Customer Insights Report (Python & Pandas)

This project performs a full customer data cleaning and analysis pipeline using Python and pandas.

## Project Objectives
- Clean and normalize customer data
- Fix inconsistent text, numeric, and date formats
- Generate business KPIs
- Identify inactive customers
- Export clean datasets and reports

## Dataset Issues Addressed
- Inconsistent name formats
- Invalid and malformed emails
- Mixed country naming conventions
- Currency symbols and mixed numeric formats
- Multiple date formats
- Customer inactivity detection

## Key Features
- Data cleaning and normalization
- Numerical transformations and KPIs
- Date handling and inactivity analysis
- Export to Excel, CSV, and TXT reports

## Technologies Used
- Python
- pandas
- datetime

## Outputs
- Clean Excel and CSV files
- Business KPI report in text format

This project is intended as a learning and portfolio project.
